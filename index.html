Box Model Lab

Each box below has a declared width of 220px, with padding and border applied. One uses content-box (default), the other border-box.

content-box (default)

  • width: 220px (content only)
  • padding: 16px
  • border: 8px
  • margin: 12px

Total visual width = 220 + 16*2 + 8*2 = 268px

border-box

  • width: 220px (includes padding + border)
  • padding: 16px
  • border: 8px
  • margin: 12px

Total visual width remains 220px (content shrinks to fit).

Tip: Setting * { box-sizing: border-box; } often simplifies layout math in real projects.
